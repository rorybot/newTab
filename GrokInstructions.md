# New Tab — product notes for Grok

Personal Brave/Chrome **new tab** extension. Not for public release — just for me.

## Tech stack

This is a **TypeScript** project. Prefer TypeScript for all extension logic.

## Inspiration

Mortality-style age clock + life segment ring.

## Current scope (v0)

- New tab override
- Precise age display + **life segment ring** (year ticks, decade emphasis)
- Optional death countdown
- Dark sandy/leather palette
- Settings via chrome.storage.local
